Information Div with Shadow Body

Information Div with Border

Rounded

Information Div with Round Corners

This info div element is for those who will want to put some curves at the edges of their information elements. It takes the class name rounded on any element which you want rounded

04

Styled Foot

Information Div with Styled Foot

This container is making use of styles to the foot element. Add the style-foot to the foot class element, and a child span with class foot-handle and an absolute positioning.

05

Styled Head

Information Div with Styled Head

This container is making use of styles to the head element. Add the style-head to the head class element, and a child span with class head-handle and an absolute positioning.

06

Styled Head n Foot

Styling Both Head and Foot

This container is making use of styles to the head and foot elements. Follow the prompts to add styles to the head and foot to achieve this
