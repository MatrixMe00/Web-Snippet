Large buttons with fade animation

Use the anim-fade class to get the fade animation effects. Add large to the span buttons to make a large button display

Medium semi-round buttons with swing animation

Use the anim-swing class to get the swing animation effects. Add med and semi-round classes to the span buttons to make a medium semi-round button display

Small round buttons with fade and swing animation

Use the anim-swing and anim-fade classes to get the swing and fade animation effects. Add small and round classes to the span buttons to make a small round button display

Very small round buttons with fade and swing animation

Use the anim-swing and anim-fade classes to get the swing and fade animation effects. Add vsmall and round classes to the span buttons to make a very small round button display

Circular loader Fade

Add circle-loader class and use the anim-fade classe to get the circular and fade animation effects. Add stroke and semi-round classes to the span buttons to make semi round stroke display
